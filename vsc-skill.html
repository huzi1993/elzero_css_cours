اختصارات VS Code مع التوضيح

1. Emmet Shortcut

Code:

ul>li*10>span.huzeyfe$

Output:

<ul> <li><span class="huzeyfe1"></span></li> <li><span class="huzeyfe2"></span></li> <li><span class="huzeyfe3"></span></li> <li><span class="huzeyfe4"></span></li> <li><span class="huzeyfe5"></span></li> <li><span class="huzeyfe6"></span></li> <li><span class="huzeyfe7"></span></li> <li><span class="huzeyfe8"></span></li> <li><span class="huzeyfe9"></span></li> <li><span class="huzeyfe10"></span></li> </ul>

Explanation:

  • EN: Creates a `ul` element with 10 `li` items, each containing a `span` with a class `huzeyfe` followed by an incremental number.
  • AR: ينشئ عنصر `ul` يحتوي على 10 عناصر `li`، كل عنصر يحتوي على عنصر `span` مع صنف باسم `huzeyfe` ورقم متزايد.

2. Delete a Line

Shortcut: Ctrl+Shift+K

Explanation:

  • EN: Deletes the current line and moves the cursor to the line above it.
  • AR: يمسح السطر الحالي وينقل المؤشر إلى السطر الذي فوقه.

3. Duplicate a Line

Shortcut: Ctrl+Shift+D

Explanation:

  • EN: Duplicates the current line or selected lines directly below the original.
  • AR: يكرر السطر الحالي أو السطور المحددة تحت النسخة الأصلية مباشرة.

4. Move a Line Up or Down

Shortcut: Alt+Up/Down Arrow

Explanation:

  • EN: Moves the current line (or selected lines) up or down in the code.
  • AR: يحرك السطر الحالي (أو السطور المحددة) لأعلى أو لأسفل في الشيفرة.

5. Select All Occurrences of a Word

Shortcut: Ctrl+Shift+L

Explanation:

  • EN: Selects all occurrences of the word at the cursor position for simultaneous editing.
  • AR: يحدد جميع تكرارات الكلمة الموجودة عند موضع المؤشر للتعديل عليها دفعة واحدة.

6. Comment/Uncomment a Line

Shortcut: Ctrl+/

Explanation:

  • EN: Toggles a comment on the current line or selected lines.
  • AR: يقوم بإضافة أو إزالة تعليق للسطر الحالي أو السطور المحددة.

7. Wrap Selection with a Tag (Emmet)

Shortcut: Ctrl+Shift+W

Explanation:

  • EN: Wraps the selected text with a specified tag.
  • AR: يلف النص المحدد داخل عنصر HTML حسب الاختيار.

8. Format Document (Prettier)

Shortcut: Shift+Alt+F

Explanation:

  • EN: Formats the entire document according to the configured formatter (e.g., Prettier).
  • AR: يعيد تنسيق المستند بأكمله وفقًا للتنسيق المحدد (مثل Prettier).
